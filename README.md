# Truck Rental Python
A menu for a truck rental 

##  Overview
This project showcases a menu that is used to help input information to get the right price for each rental. 

##  The Logic
It is a simple menu on the outside but a lot of work went into the background to make it work including but not limited to:
1. How to let python able to process upper case and lower case input as one without crashing.
2. The program works on an infinite loop which leads to finding a way to exit the program from an infinite loop.
3. How to get the right format of user input without crashing (ex: str for days, int for miles).
4. Finding every possible way to a program crashing and stopped it.
5. Making different functions for weeks and days and another functions to calculate each

## 🛠️ How to Use
1. Ensure you have Python 3 installed.
2. Run the script
3. Follow the menu instruction.
